# ISEN Weather

Project made during Android courses.

## Functionalities
	
	* Search by name
	* Search by geolocation
	* More details about city by searching it
	* Add city(ies) to your favorites
	* Last forecast saved in database for each favorite
	* Widget
	* Animations
